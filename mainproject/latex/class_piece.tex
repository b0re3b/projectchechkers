\doxysubsection{Detailed Description}
Represents a checker piece on the board. 

This class stores the state of a checker piece, including its position, color, whether it\textquotesingle{}s a king, and whether it\textquotesingle{}s still in play. It also provides functionality to draw the piece on the screen.
