\doxysubsection{Detailed Description}
Represents the main game logic and interaction. 

This class manages the core gameplay loop, including drawing the board, handling input, and facilitating turns between the human and computer players. It holds references to the game window, the board, and the player objects.

Runs the main game loop. 

This function initiates the core game loop. It continuously\+:
\begin{DoxyEnumerate}
\item Clears the window.
\item Draws the current state of the board.
\item Handles player input (from both human and computer players).
\item Updates the game state accordingly.
\item Displays the updated game state on the screen.
\end{DoxyEnumerate}

The loop continues until the game window is closed.

This function handles the continuous cycle of the game\+:
\begin{DoxyEnumerate}
\item Clears the window to prepare for rendering.
\item Draws the current state of the board onto the window.
\item Collects input from the player through the window.
\item Displays the updated window contents on the screen.
\end{DoxyEnumerate}

The loop continues until the window is closed.
